# 愿望清单小程序 - 规格文档

## 1. 项目概述

- **项目名称**: 聚了吗 - WishList & Memories
- **项目类型**: 微信小程序
- **核心功能**: 愿望清单、纪念日管理、萌宠萌娃记录、自动总结报告
- **目标用户**: 年轻群体（18-35岁），喜欢记录生活、朋友间互动频繁

## 2. UI/UX 规格

### 2.1 整体风格
- **设计风格**: 温暖治愈系，类iOS毛玻璃效果 + 柔和渐变
- **主题色**: 薰衣草紫 (#9B7DED) + 暖橙色 (#FFAB76) + 薄荷绿 (#7DD3C0)
- **字体**: 系统默认字体，iOS端 SF Pro
- **圆角**: 16rpx (小) / 24rpx (中) / 32rpx (大)
- **阴影**: 0 8rpx 24rpx rgba(155, 125, 237, 0.15)

### 2.2 页面结构

```
pages/
├── index/           # 首页 - 动态墙 Feed
├── wishlist/        # 愿望清单
├── wishlist/detail/ # 愿望详情/认领
├── wishlist/museum/ # 回忆博物馆
├── anniversary/     # 灵动纪念日
├── moment/          # 萌宠/萌娃墙
├── moment/profile/  # 宠物/娃档案
├── capsule/         # 时间胶囊年度报告
├── profile/         # 个人中心
└── create/          # 创建页面（发愿望/晒宠物/设纪念日）
```

### 2.3 页面详细设计

#### 2.3.1 首页 - 动态墙 (Feed)
- **顶部**: 渐变背景 + 当日天气插画 + 日期
- **卡片流**: 瀑布流布局，卡片高度自适应
- **卡片内容**:
  - 用户头像 + 昵称 + 时间
  - 内容类型图标（愿望/纪念日/萌宠）
  - 内容缩略展示
  - 底部: 点赞按钮 + 评论弹幕入口
- **底部导航**: 固定 TabBar (首页/愿望/纪念日/我的)

#### 2.3.2 愿望清单页面
- **顶部筛选**: 标签栏（全部/进行中/已达成/回忆博物馆）
- **愿望卡片**:
  - 分类图标（餐厅🍽️/旅行✈️/运动⚽/其他💫）
  - 愿望标题
  - 认领状态: "发起人: xxx | 已认领: xxx/3"
  - 进度条（已打卡/总天数）
  - 操作按钮: 认领/打卡/点亮
- **FAB**: 右下角 "+" 按钮，发起新愿望

#### 2.3.3 愿望详情/认领页
- **大图轮播**: 愿望相关图片
- **愿望信息**: 标题、描述、分类、目标日期
- **认领列表**:
  - 显示所有认领者
  - 每人可勾选"我想去"
  - 打卡记录
- **点亮入口**: 达成后显示"点亮此愿望"按钮

#### 2.3.4 回忆博物馆
- **时间轴**: 垂直时间轴，按达成时间排序
- **回忆卡片**:
  - 愿望内容
  - "点亮"时的现场照片（1-3张）
  - 参与人员tag
  - 情感标签

#### 2.3.5 灵动纪念日
- **分类 Tab**: 固定日期 / 周期提醒
- **固定日期卡片**:
  - 名称 + 日期
  - 倒计时显示（XX天XX时XX分）
  - 农历/公历切换
- **周期提醒卡片**:
  - 图标 + 名称
  - 下次提醒时间
  - 开启/关闭开关

#### 2.3.6 萌宠/萌娃墙
- **顶部切换**: 萌宠 / 萌娃
- **档案入口**: 头像卡片，点击进入档案页
- **时光轴**: 按发布时间排列的照片流
- **照片卡片**:
  - 单图/九宫格展示
  - 发布时间 + 年龄标记
  - 互动: 投喂罐头🥫/爱心💗

#### 2.3.7 宠物/娃档案页
- **头部**: 大头像 + 名字 + 性格关键词
- **里程碑时间轴**:
  - 第一次会爬 🐾
  - 第一次洗澡 🛁
  - 第一次...（自定义）
- **成就徽章**:
  - 万众宠爱（收到100+罐头）
  - 成长记录（发布50+照片）
  - 等...

#### 2.3.8 时间胶囊年度报告
- **首页动画**: 下拉刷新触发年度报告生成动画
- **页面结构**:
  - 封面: "2024年度回顾"
  - 词云: 愿望中最高频的词（视觉化云图）
  - 勤奋榜: 好友排行榜（晒娃/晒宠最多）
  - 陪伴值: 共同打卡最多的好友组合
  - 随机回忆: "去年的今天，你们一起完成了..."
  - 生成图片: 长按保存分享

#### 2.3.9 创建页面
- **顶部选择器**: 发愿望 / 晒宠物 / 设纪念日
- **表单字段**:
  - 愿望: 标题 + 描述 + 分类 + 目标日期 + 图片上传
  - 纪念日: 名称 + 日期 + 类型（固定/周期）+ 提醒开关
  - 萌宠: 照片 + 文字 + 投喂选项

## 3. 功能规格

### 3.1 愿望清单 (Wishlist)

#### 3.1.1 认领模式
- 创建愿望时可设置"允许多人认领"
- 认领时显示"我想去"勾选框
- 发起人可查看认领者列表
- 支持查看认领者详情

#### 3.1.2 进度可视化
- 愿望设置目标日期
- 实时显示: 已进行X天 / 剩余X天
- 达成条件: 目标日期到达 or 手动标记完成
- "点亮"操作:
  - 触发回忆博物馆录入流程
  - 上传1-3张现场照片
  - 填写回忆感想

#### 3.1.3 统计维度
- **时间轴视图**: 按创建时间排序，可筛选月份
- **分类统计**:
  - 餐厅（聚餐、探店）
  - 旅行（出游、远行）
  - 运动（健身、户外）
  - 其他

### 3.2 灵动纪念日 (Anniversary)

#### 3.2.1 固定日期提醒
- 支持公历/农历切换
- 倒计时显示: 生日/纪念日提前30天开始倒计时
- 推送通知: 提前7天/3天/当天提醒

#### 3.2.2 周期提醒
- 支持自定义周期（每周X/每月X）
- 预设模板: 疯狂星期四、周五电影夜
- 开关控制: 可单独开启/关闭

### 3.3 萌宠/萌娃墙 (Moment Wall)

#### 3.3.1 档案系统
- 基础信息: 名字、头像、品种/年龄
- 性格关键词: 最多5个标签
- 重要里程碑:
  - 预设模板: 第一次会爬、第一次走路、第一次洗澡
  - 自定义添加

#### 3.3.2 互动机制
- **虚拟投喂**:
  - 罐头🥫 / 爱心💗 按钮
  - 数字累加显示
  - 解锁条件:
    - 万众宠爱: 100+罐头
    - 超级明星: 500+爱心
- **成长曲线**:
  - 自动按发布时间生成时间轴
  - 显示年龄里程碑

### 3.4 自动总结 (Time Capsule)

#### 3.4.1 年度报告
- 入口: 个人中心 - 年度报告
- 视觉: 长图模式，类Spotify年度听歌报告风格

#### 3.4.2 核心指标
- **年度/月度词云**: 统计愿望清单中出现最多的词
- **勤奋榜**: 好友中发布萌宠/娃照片最多的TOP3
- **陪伴值**: 与好友共同参与愿望打卡的总次数
- **随机回忆**: "去年的今天"模块，调取往年同天达成的愿望

### 3.5 动态墙 (Feed)

#### 3.5.1 自动同步
- 触发条件:
  - 新建愿望
  - 愿望被认领
  - 愿望达成"点亮"
  - 新增纪念日
  - 发布萌宠/娃照片
- 生成简讯格式:
  - "[用户] 发起了新愿望: [标题]"
  - "[用户] 点亮了愿望: [标题]"
  - "[用户] 晒出了 [宠物名] 的可爱瞬间"

#### 3.5.2 评论弹幕
- 图片下方显示弹幕输入框
- 弹幕特点:
  - 最多显示20条最新弹幕
  - 3秒后渐隐
  - 可点赞弹幕

## 4. 交互流程

### 4.1 核心流程
```
1. 登录 → 首页（动态墙）
2. 点击"+" → 选择类型（愿望/纪念日/萌宠）
3. 填写表单 → 提交 → 触发推送通知好友
4. 周期性 → 每月1号生成月报
```

### 4.2 通知机制
- 群机器人: 微信群内通知
- 订阅消息: 服务通知

## 5. 技术实现

### 5.1 框架
- 微信小程序原生开发
- 使用 uni-ui 组件库（可选）
- 数据存储: 本地缓存 + 云开发（可选）

### 5.2 页面栈
- TabBar: 4个固定页面
- Navigator: navigateTo 跳转

### 5.3 动画
- CSS动画: 基础过渡
- 小程序动画API: 复杂动效
